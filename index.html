How it works

1
Capture intent

2
Compare odds

3
Route & convert

Integration tiers

Start fast, then deepen integration over time.

Tier 1
Deep link

Fastest time-to-live. Route into operator apps/sites with tracking and event context.

Tier 2
Webview journey

Embedded operator experience where available, improving continuity while staying pragmatic.

Tier 3
Full API (login/registration/bet placement)

Deepest capability. Enables true server-side journeys for selected operators.
